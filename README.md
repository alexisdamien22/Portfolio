# Portfolio Alexis Damien

Ce projet est un portfolio web dynamique développé en PHP. Il utilise le moteur de template Twig, une architecture MVC simplifiée et une intégration avec l'API GitHub pour récupérer automatiquement les projets.

## 🚀 Installation

### Prérequis
* PHP 8.1 ou supérieur
* [Composer](https://getcomposer.org/) installé sur votre machine
* Un serveur local (ou `php -S` intégré)
